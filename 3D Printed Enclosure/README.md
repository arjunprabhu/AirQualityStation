# 3D Printed Enclosure

The custom 3D printed enclosure for the Air Quality Station provides a sleek, desktop-friendly housing that securely holds the Seeed XIAO ESP32-S3, the ILI9341 display, and all the I2C sensors. It includes proper ventilation for the particulate and gas sensors, as well as a precise cutout for the mmWave radar.

## Download 3D Models

To ensure you are getting the most up-to-date versions of the enclosure, the STL files and the complete Fusion 360 source file (`.f3d`) have been uploaded to **MakerWorld**.

### What's included on MakerWorld:
- **STL Files:** Ready-to-print parts for the front bezel and rear housing.
- **Bambu Studio 3MF File:** Pre-sliced profiles with optimal support placements.
- **Fusion 360 Source File:** Fully parametric source file so you can modify the enclosure to fit different displays, cables, or additional sensors.

## Printing Recommendations

- **Material:** PETG or PLA is fine, but PETG is recommended if the unit will sit in direct sunlight.
- **Resolution:** 0.20mm layer height works great.
- **Supports:** Required for the sensor cutouts and the USB port opening.
